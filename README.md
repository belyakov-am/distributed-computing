# distributed-computing
## Description
В данной ветке лежат первые три дз с некоторыми бонусами (пагинация, докер, бд)

## Deploy
`docker-compose up --build`

## Project description
В данной ветке представлена реализация базового апи для магазина. Весь проект написан на питоне с использованием Django и Django Rest Framework. В качестве бд выбор пал на PostgreSQL. Все возможные запросы описаны в постман схеме.
